# ⌨️ Virtual Keyboard (Vue)

A simple virtual keyboard built with Vue.js. This project simulates a real keyboard, allowing users to type using on-screen keys with a smooth and interactive experience.

## 🚀 Demo

Check out the live demo:  
👉 https://maxwellolliver.github.io/virtual-keyboard/

## ✨ Features

- On-screen virtual keyboard
- Interactive key press animations
- Responsive layout
- Real-time typing display
- Support for basic keyboard inputs

## 🛠️ Tech Stack

- Vue.js
- HTML5
- CSS3

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/maxwellolliver/virtual-keyboard.git

# Navigate into the project
cd virtual-keyboard

# Install dependencies
yarn

# Run the development server
yarn dev
